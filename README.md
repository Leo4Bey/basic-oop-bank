# Bank Account Management System

This is a simple C++ application demonstrating a basic bank account management system. It allows users to perform operations such as depositing and withdrawing money from a bank account.

## Features

- **Deposit Money**: Add money to the account balance.
- **Withdraw Money**: Withdraw money from the account balance.
- **Check Balance**: View the current account balance.

## Files Included

- `BankInfo.cpp`: Contains the implementation of the `BankaHesabi` class, which handles the bank account operations.
